Intervals

Published on June 12, 2023

Given the real numbers a and b with a < b, we define:


a) Open interval with endpoints a and b is the set

\[ ]a, b[ = \{ x \in \mathbb{R} \mid a < x < b \} \]

This expression means it is the set all real numbers between a and b excluding the endpoints a and b themselves. For example:

\[ ]1, 4[ = \{ x \in \mathbb{R} \mid 1 < x < 4 \} \]

The example shows an interval whose set includes {1.0000000......n, 1.25, √5, π, 3.999999......n}, but does not include 1 and does not include 4. (The n in the set indicates that it extends to the nth decimal place)

b) Closed Interval with endpoints a and b is the set

\[ [a, b] = \{ x \in \mathbb{R} \mid a \leq x \leq b \} \]

This expression means it is the set that includes all real numbers between a and b including the endpoints a and b themselves. For example:

\[ [2, 5] = \{ x \in \mathbb{R} \mid 2 \leq x \leq 5 \} \]

The example shows an intrval whose set includes { 2, π, 3, 4.999.........n, 5}, including 2 and including 5.

c) Left-closed (or right open) interval with endpoints a and b is the set:

\[ [a, b[ = \{ x \in \mathbb{R} \mid a \leq x < b \} \]

This expressiona means it is the set that includes a but not b. for example:

\[ [1, 4[ = \{ x \in \mathbb{R} \mid 1 \leq x < 4 \} \]

The example shows an interval whose set includes {1, 2.5, π, 3.9999....n}, including 1 but not including 4.

d) Right-closed (or left-open) interval with endpoints a and b is the set:

\[ ]a, b] = \{ x \in \mathbb{R} \mid a < x \leq b \} \]

This expression means it is the set that does not include a but include b. For example:

\[ ]2, 5] = \{ x \in \mathbb{R} \mid 2 < x \leq 5 \} \]

The example shows an interval whose set includes {2.0000.....n, 2.5, π, 3.9999....n, 4, 5} excluding 2 but including 5.

Within this context, a is referred to as the lower endpoint of interval, and b as the upper endpoint of interval.

We also consider "infinite intervals" defined as follows:

e) Left-infinite type:

\[ ]-\infty, a[ = \{ x \in \mathbb{R} \mid x < a \} \]

The symbol \( -\infty \) indicates that there is no lower bound, meaning it extends infinitely to the left. It is always open on the infinite side, as infinity is never included, in this context it is just a symbol of unlimited growth. Example:

\[ ]-\infty, 5[ = \{ x \in \mathbb{R} \mid x < 5 \} \]

The set { \( -\infty \), -10, -0.5, π, 4.999999........n}, includes all numbers less than 5 up to infinity, but not 5 itself.

Observação: Em algumas literaturas é escrito como:

\[ (-\infty, a[ \]
ou
\[ -\infty \longrightarrow a \]

I won’t show how it is written in all types of literature, as it would confuse more than help.

f) Left-infinite type with a included:

\[ ]-\infty, a] = \{ x \in \mathbb{R} \mid x \leq a \} \]

It goes from negative infinity up to a, including it. Example:

\[ ]-\infty, 5] = \{ x \in \mathbb{R} \mid x \leq 5 \} \]

The set { \( -\infty \), -100, 0, 3, 4.99999.....n, 5} goes from negative infinity up to 5, including 5.

g) Right-infinite type:

\[ ]a, +\infty[ = \{ x \in \mathbb{R} \mid x > a \} \]

This is the set of all real numbers greater than a, meaning a is not included in the set. Example:

\[ ]3, +\infty[ = \{ x \in \mathbb{R} \mid x > 3 \} \]

The set includes {3.0000......n, 4, 500, 1000000, \( +\infty \)}, but not including 3.

h) Right-infinite with a included:

\[ [a, +\infty[ = \{ x \in \mathbb{R} \mid x \geq a \} \]

This represents the set of all real numbers greater than or equal to a. Example:

\[ [3, +\infty[ = \{ x \in \mathbb{R} \mid x \geq 2 \} \]

The example shows a set that includes {3, 3.9999.....n, , 4, 500, 1000000, \( +\infty \)}, including 3.

i) Open interval of Real numbers:

\[ ]-\infty, +\infty[ = \mathbb{R} \]

This represents all real numbers, ranging from negative infinity to positive infinity, without including them.
